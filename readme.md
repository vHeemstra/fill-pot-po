# fill-pot-po

## Information

| Package     | wp-pot                                               |
| ----------- | ---------------------------------------------------- |
| Description | Generate pre-filled PO files from POT file, using source PO files. |

## Install

```sh
npm install --save-dev fill-pot-po
```
